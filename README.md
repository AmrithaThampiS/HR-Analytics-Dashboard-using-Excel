# HR-Analytics-Dashboard-using-Excel
An end-to-end Data Analytics project built using Microsoft Excel, focused on analyzing and visualizing Human Resource (HR) data to support better decision-making.
📌 Project Overview

The HR Analytics Dashboard provides a comprehensive view of employee data, helping organizations monitor key HR metrics such as attrition, workforce distribution, and employee demographics.

It transforms raw HR data into meaningful visual insights that support data-driven decision-making.
